## 1. Circle Detection (`GaugeDetector::detectCircle`)
- `GaugeDetector` class owns `GaugeROI` (center + radius) and `ScaleCalibration` internally
- Tries 7 HoughCircles parameter sets (dp, canny, acc) *with* Gaussian blur (9Ă—9, Ïƒ=2)
- If all fail, retries the same 7 sets *without* blur

- Radius bounds: `[maxDim/15, maxDim/2]`

## 2. Interactive Calibration
- Single ImGui/GLFW window "Gauge Reader" â€” video displayed as OpenGL texture via `ImGui::Image`
- **Phase 0**: user clicks minimum-value marking on the image
- **Phase 1**: user clicks maximum-value marking on the image
- **Phase 2**: two ImGui sliders (0â€“1000 range); press Confirm button

- Angles from clicks â†’ `calibrateFromPoints()` computes `startAngle`/`endAngle` (via `atan2`)
- Slider values â†’ `setCalibrationValues()` sets `minValue`/`maxValue`

## 3. Needle Detection (`GaugeDetector::detectNeedle`)
Two methods, tried in order:

### a) Colored needle (`detectColoredNeedle`)
- HSV segmentation for red (two hue ranges: 0â€“10 and 160â€“179)
- Morphological close + open to clean mask
- Find external contours, filter by area (> radius/2) and centroid distance (< 0.6Ă—radius)
- Score contours by `maxTipDistance Ă— log(area)`
- Return angle from center to farthest contour point

### b) Radial scan fallback (`detectNeedleRadial`)
- Adaptive threshold (Gaussian, block=25, C=8) on masked grayscale
- 360 radial lines from 8% radius to edge
- Per angle: scan outward, track longest contiguous dark run + total dark density
- Score = `density Ă— 0.4 + longestRun/radius Ă— 0.6`
- Needle has long continuous dark run; tick marks produce only short peripheral dark segments

## 4. Angle-to-Value Mapping (`GaugeDetector::angleToValue`)

## 5. Output Smoothing
- 5-frame deque of recent values

## Key Architecture
- Single GLFW window, Dear ImGui for all UI (no OpenCV highgui)
- VideoTexture struct: uploads `cv::Mat` frames as OpenGL textures via `glTexImage2D`
- `tryGetClickOnImage()`: converts ImGui `Image` click position back to pixel coords
- Calibration uses ImGui sliders (0â€“1000 range, raw), not OpenCV trackbars
- `GaugeDetector` class encapsulates all detection logic; owns `GaugeROI` and `ScaleCalibration` internally
- GUI works with `detector.gauge()` / `detector.scale()` const accessors; calibration set via methods
